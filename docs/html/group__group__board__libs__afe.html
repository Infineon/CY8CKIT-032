◆ cy8ckit_032_init()

cy_rslt_t cy8ckit_032_init ( cyhal_i2c_t *  inst)

Initializes the I2C interface for communicating with the AFE and OLED components on the board.

Parameters
[in]instAn optional I2C instance to use for communicating with the peripherals on the shield. If NULL, a new instance will be allocated using the CYBSP_I2C_SCL & CYBSP_I2C_SDA pins.
Returns
CY_RSLT_SUCCESS if properly initialized, else an error indicating what went wrong.
